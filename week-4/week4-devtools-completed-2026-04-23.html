task-03

Understanding the box model

✓ Complete

Every element on a page is a box. The CSS Box Model has four layers:

  • Content — the text, image, or other content
  • Padding — space around content, inside the border
  • Border — a line around the padding
  • Margin — space outside the border, between elements

Different color formats

✓ Complete

CSS lets you write colors in several formats:

  • Named — red, coral, tomato
  • HEX — #FF6B6B
  • RGB — rgb(255, 107, 107)
  • HSL — hsl(0, 100%, 71%)
  1. Inspect the Red swatch from Task 7.
