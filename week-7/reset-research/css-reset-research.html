Eric Meyer — Reset Reasoning

Main Idea:

Eric Meyer created CSS resets to reduce browser inconsistencies and create a cleaner starting point for developers.

What stood out:

How much time developers used to have to spend fighting browser defaults because they were different from browser to browswer.

Josh Comeau — A Modern CSS Reset

Main Idea:

Modern resets focus more on accessibility, usability, and responsive design instead of removing everything.

What stood out:

Modern browsers are much more consistent now. Css support has improved a lot, and standards are more reliable now.

Critical Questions

Why reset CSS?

Resets reduce inconsistencies and create a more consistent and predictable starting point. They also remove or fix conflicting browser default styles giving more control over design and layout.

Why use Josh’s version over Eric’s?

  • Preserves useful defaults
  • Better accessibility features
  • Modern responsive techniques
